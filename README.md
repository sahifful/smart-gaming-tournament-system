# \# Smart Gaming Tournament Analytics System

# 

# This project is developed for BIT1034 Advance Programming.

# 

# \## Project Description

# The Smart Gaming Tournament Analytics System is a Streamlit web application that helps tournament organizers manage player data, upload CSV datasets, store data in SQLite database, generate automatic rankings, and display analytics charts.

# 

# \## Technologies Used

# \- Python

# \- Streamlit

# \- SQLite

# \- Pandas

# \- Matplotlib

# 

# \## Main Features

# \- CSV dataset upload

# \- SQLite database storage

# \- Player records display

# \- Automatic ranking generation

# \- Dashboard analytics charts

# 

# \## How to Run

# 1\. Install the required libraries:

# &#x20;  pip install -r requirements.txt

# 

# 2\. Run the application:

# &#x20;  streamlit run app.py
